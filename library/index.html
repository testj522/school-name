Print Journal Search

Are you looking for a specific journal title? Search the Catalog to find which journals, magazines or periodicals in our print collection and which individual volumes we own.

If you need to search the contents of journals, use a database listed in A-Z Electronic Resources.

Trouble with Off-Campus Access

We use EZproxy to authenticate access to GALILEO databases, electronic journals, eBooks, and other licensed electronic resources from off campus. You will be prompted to login using your SWAN username and password. You will only need to login once when accessing multiple resources during any one session. However, if you close out of your browser and then re-enter it, you will need to go through the login process again.

Use the Off-Campus Access to Electronic Resources guide for help with signing in from off campus.
